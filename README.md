# PyTerminal-OS: A Mini CLI Computer

**PyTerminal-OS** is a Python-based mini-operating system environment that runs entirely within the Command Line Interface (CLI). It simulates a personal computer experience by grouping essential tools, utilities, and games under a single secured dashboard.

---

## ğŸ–¥ï¸ About the Project
The core idea of this project is to provide a "Mini PC" experience. Instead of running separate scripts, the user logs into a secure environment and has access to a suite of built-in applications, ranging from productivity tools to entertainment.

### Built-in Applications:
1.  **Ping Pong Game**: A retro-style arcade game.
2.  **Digital Clock**: A live-tracking digital clock.
3.  **YouTube Downloader**: A utility to download videos directly via URL.
4.  **Notepad**: A lightweight text editor for quick notes.
5.  **Student Portal**: A dedicated management interface for university students.
